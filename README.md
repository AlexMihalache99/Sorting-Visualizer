# Sorting-Visualizer

I created a tool for visualizing various sorting algorithms.<br>
You can find the project at [Sorting Visualizer](https://alexmihalache99.github.io/Sorting-Visualizer/).

- Insertion sort <br>
- Heap sort <br>
- Quick sort

- CSS <br>
- Bootstrap 5 <br>

- jQuery
